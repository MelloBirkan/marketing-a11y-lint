# A11Y Lint - Marketing Site

This is the marketing website for the A11Y Lint TCC project. Built using [Tailwind CSS](https://tailwindcss.com) and [Next.js](https://nextjs.org).

## Getting started

To get started with this project, first install the npm dependencies:

```bash
npm install
```

Next, run the development server:

```bash
npm run dev
```

Finally, open [http://localhost:3000](http://localhost:3000) in your browser to view the website.

## Customizing

You can start editing the marketing site by modifying the files in the `/src` folder. The site will auto-update as you edit these files.
